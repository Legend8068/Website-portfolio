# Ayush Singh - Portfolio Website

Welcome to the repository for my personal portfolio website! 

This is an interactive and dynamic single-page web application designed to showcase my experience, projects, and skills as an undergraduate Computer Science & Design student at the Singapore University of Technology and Design (SUTD).

## 🌟 Key Features

The portfolio is designed with a focus on high-performance creative coding, modern aesthetics, and seamless user experiences.

- **Creative Scrolling Transitions**: A custom scroll-driven SVG portal animation that transitions the user smoothly from the Hero section into the Experience/Projects section.
- **Dynamic Geometric Backgrounds**: A cursor-reactive 2D waveform that organically dissolves into an intricate node network as you scroll down the page, created using canvas.
- **Decrypt Text Effects**: Matrix-style decrypted text reveals on headings for a cyberpunk-inspired technical feel.
- **Responsive Layout**: Fluid typography and spacing utilizing CSS `clamp()` to ensure beautiful rendering across all devices from mobile to ultrawide desktops.

## 💻 Tech Stack

- **Framework**: React 18+ (Vite)
- **Styling**: Inline JavaScript styles for dynamic rendering and performance (No external CSS frameworks required)
- **Icons**: Lucide React
- **Animations/Graphics**: HTML5 `<canvas>`, inline SVG masking, and native CSS transitions/animations
- **Deployment**: Configured for Vercel/Netlify environments

## 🚀 Running Locally

To run this project on your local machine:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Legend8068/Website-portfolio.git
   ```

2. **Navigate to the directory**
   ```bash
   cd Website-portfolio
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```
   *(Note: ensure you have Node.js installed)*

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Open in browser**
   Navigate to `http://localhost:5173` in your web browser.
